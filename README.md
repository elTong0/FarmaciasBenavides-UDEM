# Proyecto: Farmacias Benavides #

En este repositorio podrá encontrar todos los artefactos desarrollados como parte del proyecto elaborado para Farmacias Benavides, incluyendo documentación, historias de usuario y componentes funcionales del sistema.

## Contenido del repositorio ##
-Archivos .xlsx: **contienen varias hojas (pestañas), donde se encuentran documentados todos los Sprint Backlogs y sus respectivas Historias de Usuario**, de acuerdo con la planificación y desarrollo del proyecto.

-Carpeta **farmacias_benavides/:**  contiene la implementación de la función de Antecedentes Médicos, la cual permite:

-Consultar antecedentes médicos registrados.

-Crear nuevos antecedentes médicos para pacientes.

-Editar o actualizar información existente.
